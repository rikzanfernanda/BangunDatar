# Bangun Datar
Library yang berisi rumus untuk menyelesaikan perhitungan bangun datar:
* Persegi
* Persegi panjang
* Belah ketupat
* Jajargenjang
* Layang-layang
* Lingkaran
* Segitiga
* Trapesium

## How to Install
```bash
composer require fernanda/bangundatar
```
Atau clone
```bash
git clone https://github.com/rikzanfernanda/bangundatar.git
```

## How to Use
* Persegi
use BangunDatar\Persegi;
